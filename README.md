# Portfólio Dev FullStack (React)

## Um portfólio moderno e responsivo.
### Recursos incluídos:

- Navegação one-page com scroll suave
- Seção de projetos filtrados por tecnologia
- Visualizador de currículo em PDF
- Formulário de contato integrado (Formspree)
- 100% mobile-responsive
- Suporte a múltiplos idiomas (PT-BR/EN) (Em breve)

## ? Tecnologias

- Frontend: React, Vite, TailwindCSS
- Hospedagem: Vercel
